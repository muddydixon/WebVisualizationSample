エンジニアのための データ可視化[実践]入門
~D3.jsによるWebの可視化 サンプルプログラム~

D3を利用して可視化を作成するためのひとつのサンプルとして、初学者のみなさまの役に立てていただければと思いサンプルのコードと実際の可視化を公開します

データには、Rから取得したもの、川崎市の公開データから採取したもの、国立情報学研究所/ドワンゴが提供してくださったニコニコ動画を含みます

閲覧方法

下記のいずれかの方法でWebサーバを立て、ご確認ください

  1. Apache: documentRootの下においてください
  2. Python 3.3 
    $ python -m http.server  8888
  3. Python 2.7 
    $ python -m SimpleHTTPServer 8000
  4. Ruby(WebRick) 
    $ ruby -rwebrick -e 'WEBrick::HTTPServer.new(:DocumentRoot => "./", :Port => 8000).start'
  5. Node 
    $ node -e 'var fs=require("fs"), path = require("path"); require("http").createServer(function(req, res){ url = path.join(__dirname, req.url); res.end(fs.readFileSync(url.replace(/\/$/, "/index.html"), "utf8"))}).listen(8000)'
